# Glossário

## Builder

Conjunto de telas do painel para criar rotas e montar páginas dinâmicas.

## Snapshot publicado

Arquivo JSON que a loja consome em runtime, sem depender da estrutura administrativa interna.

## Template do storefront

Documento estrutural com tema, header, home, footer e mega menu.

## Namespace operacional

Convencao de caminho usada para classificar páginas por objetivo, como `landing` ou `institucional`.

## Runtime de servidor

Modo em que o storefront resolve conteúdo dinâmico com o processo do Next ativo.

## Export estático

Modo em que o app foi exportado como artefato fixo. Novas rotas exigem nova exportação.

## Espelho legado

Arquivo agregado mantido por compatibilidade, mesmo quando a fonte principal já foi quebrada em documentos menores.
