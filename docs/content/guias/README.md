# Guias Técnicos

Esta pasta foi pensada para leitura explicativa, em tom de conversa técnica.

Cada nota tenta responder quatro perguntas:

- o que esse módulo faz;
- qual é o trecho de código que sustenta a ideia;
- qual é a ordem de execução;
- como explicar isso de forma clara em documentação ou treinamento.
