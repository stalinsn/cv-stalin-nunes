# Mock Catalog (Supermarket)

This folder contains a generated mock catalog with 100 product JSON files.

## Structure
- products/: one product per JSON file
- manifest.json: index with all files and basic metadata

## Notes
- Files are mock data for future admin-panel-driven catalog integration.
- Product schema includes legacy ecommerce fields plus extended attributes (pricing, stock, logistics, SEO, regionalization, variationGroup).
